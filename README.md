# نظام تقييم الموظفين - Employee Evaluation System

نظام شامل لإدارة الاستمارات الديناميكية وتقييم الموظفين

## المميزات

- **استمارات ديناميكية**: إنشاء استمارات مخصصة بسهولة
- **حقول متعددة**: دعم أنواع مختلفة من الحقول (نص، ملف، قائمة منسدلة، إلخ)
- **إدارة الإدارات**: تنظيم الموظفين حسب الإدارات
- **نظام صلاحيات**: أدوار متدرجة (مدير عام، مدير، مشرف)
- **رفع ملفات آمن**: نظام حماية متقدم للملفات
- **تقارير شاملة**: إحصائيات وتقارير مفصلة

## التثبيت

### 1. نسخ المشروع
```bash
git clone <repository-url>
cd employee-evaluation-system
```

### 2. تثبيت التبعيات
```bash
composer install
```

### 3. إعداد قاعدة البيانات
```bash
# إنشاء قاعدة البيانات
mysql -u root -p -e "CREATE DATABASE employee_evaluation_system;"

# تشغيل Migration
mysql -u root -p employee_evaluation_system < database/migrations/2024_01_01_000000_create_employee_evaluation_system_tables.sql
```

### 4. إعداد الصلاحيات
```bash
chmod -R 755 storage/
chmod -R 644 storage/forms/.htaccess
```

### 5. إعداد التكوين
- نسخ ملف التكوين النموذجي
- تحديث إعدادات قاعدة البيانات
- إعداد مفاتيح التشفير

## الاستخدام

### تسجيل الدخول الافتراضي
- **البريد الإلكتروني**: admin@example.com
- **كلمة المرور**: admin123

⚠️ **هام**: يجب تغيير كلمة المرور الافتراضية فوراً في الإنتاج!

### إنشاء استمارة جديدة
1. تسجيل الدخول كمشرف
2. الانتقال إلى قسم "الاستمارات"
3. النقر على "إنشاء استمارة جديدة"
4. إضافة الحقول المطلوبة
5. حفظ ونشر الاستمارة
